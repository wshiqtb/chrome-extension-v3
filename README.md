# 原型展示插件
新版 原型展示 chrome扩展插件
[官网地址](http://axhub.qianxin-inc.cn/)

## 开发文档
```
# 装包
yarn

# 启动
yarn serve
```

最后把dist文件夹导入到chrome扩展里

## 发布
```
yarn build
```
把生产的artifacts目录下的zip包下载解压，导入到chrome扩展中
